בפרק הזה ביומן שלי אכתוב על מסעותיי ברחבי רשת ZeroNet. הכתבה הזאת היא חלק מסדרת כתבות בנושא ה-deep web. ההקדמה נמצאת בקישור הבא: הצד האחר של הרשת - הקדמה

ZeroNet

הצד האחר של הרשת - הקדמה

את ה-ZeroNet אני מריץ על Docker. הנה התמונה שמצאתי ב-Docker Hub:

Docker Hub

https://hub.docker.com/r/nofish/zeronet

https://hub.docker.com/r/nofish/zeronet

משכתי את התמונה:

$ docker pull nofish/zeronet

יצרתי תיקיה חדשה ונכנסתי אליה:

$ mkdir zeronet && cd zeronet

הרצתי קונטיינר שירוץ ברקע ויימחק ברגע שיעצור והגדרתי שה-zeronet יעבוד מול Tor שמותקן בקונטיינר:

$ docker run -d --rm -e "ENABLE_TOR=true" -v $PWD:/root/data -p 15441:15441 -p 127.0.0.1:43110:43110 nofish/zeronet

פתחתי את הדפדפן chromium אותו הקשחתי:

$ chromium-browser --incognito http://127.0.0.1:43110/

http://127.0.0.1:43110/

אגב, בהמשך החלפתי ל-Firefox מוקשח.

* קיבלתי שגיאה:

you are not going to set up a public gateway. however, your web ui is open to the whole internet. please check your configuration.

בכל מקרה, הצלחתי להוריד מידע.

הרצתי הפעם ללא חיבור ל-Tor:

$ docker run -d --rm -v $PWD:/root/data -p 15441:15441 -p 127.0.0.1:43110:43110 nofish/zeronet

השגיאה הקודמת כבר לא מופיעה אבל כמובן שב-notification שמראה על חיבור ל-Tor יש Error. נראה לי שאפשרי לעבוד בלי Tor, כל עוד אנונימיות וכולי לא מעניינות אותי. אני צריך לבדוק את העניין של השגיאה ואת העניין של גלישה ללא Tor.
